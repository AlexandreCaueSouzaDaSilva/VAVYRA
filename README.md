## VAVYRA PROJECT

# Vavyra é um gerenciador de tarefas inteligente desenvolvido em Python, com foco na otimização de tempo para profissionais. O sistema cria tarefas, adiciona, visualiza e aplica um cronograma adaptado, ajudando na organização da rotina e aumento da produtividade.




#Ferramentas:

Pandas (Biblioteca) - responsável por organizar dados em tabelas.
Datetime (Módulo Nativo do Python) - calcular datas até a prova
matplotlib (Biblioteca) → gera gráficos visuais


# Instalação
- Clone o repositório: git clone https://github.com/AlexandreCaueSouzaDaSilva/VAVYRA.git
- Instale as dependências: pip install -r requirements.txt
- Execute o projeto: python main.py


# Estrutura do Projeto:
    src/ - código-fonte principal
    docs/ - documentação
    requirements.txt - dependências
    README.md → apresentação geral


# Funcionalidades:
- Geração de cronogramas personalizados
- Cálculo automático de prazos
- Visualização gráfica de tarefas e métricas
- Exportação de dados para análise


# Planos futuros:
- Flask ou Streamlit (versão 2.0 do projeto) - transformar em app web
- Integração com calendários externos e notificações inteligentes
- Sincronização de dados em nuvem
